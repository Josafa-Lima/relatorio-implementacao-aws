# Redução de Custos em Farmácias com AWS

Projeto focado na implementação de serviços AWS com objetivo de otimizar custos operacionais em uma rede de farmácias.

## Serviços Utilizados

- Amazon EC2 com Auto Scaling
- Amazon S3 Intelligent-Tiering
- AWS Cost Explorer

## Objetivo

Aplicar boas práticas do AWS Well-Architected Framework,
com foco no pilar de Otimização de Custos.

## Resultados Esperados

- Redução de até 35% nos custos de infraestrutura
- Monitoramento financeiro contínuo
- Infraestrutura escalável
